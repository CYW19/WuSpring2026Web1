FINAL PROJECT PROPOSAL

PETS

Theme

My website is an interactive pet gallery. Users can click on photos of different animals to learn more about each pet. The purpose of my website is to help people learn about various pets.

Website Description

The website will feature pet photos, animal profiles, animations, interactive buttons, and hover effects. When users click on a pet photo, information about that animal will appear on the screen.

Fonts

I want to choose lively, friendly fonts because they reflect the cute animals on the website. I plan to use Oswald for the headings and Playfair Display for the body text.

Colors

I want to use soft pastel shades like white, light yellow, light pink, and light brown to convey the warm, comforting feeling that pets bring.

JavaScript

I plan to use JavaScript to create clickable pet cards, image animations, and animations that activate when the user clicks on them.

p5.js Element

I want to use p5.js to create animated paw prints and a background with twinkling stars. The user's mouse clicks will trigger these animations.

jQuery Element

I plan to use jQuery functions such as slideToggle(), fadeIn(), and hover effects to display pet information and create the animation effects I want.
